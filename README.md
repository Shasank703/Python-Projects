# 🎯 Number Guessing Game

## 📖 Overview

The Number Guessing Game is a Python console application where the player tries to guess a randomly generated number within a limited number of attempts. The game provides hints after each incorrect guess and keeps track of the player's score.

This project was built to strengthen my understanding of Python fundamentals, including loops, conditions, functions, lists, and the `random` module.

---

## ✨ Features

- 🎲 Random number generation
- 🔢 Number range from 1 to 100
- 💯 Score tracking
- 🚫 Duplicate guess detection
- 📉 Limited number of attempts
- 💡 Higher/Lower hints
- 🔄 Play Again option
